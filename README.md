## Introduction

This repository contains a collection of code examples and exercises that cover the fundamental concepts of core Java. Each topic is explained with code snippets and exercises to help reinforce your learning.

## Getting Started

To get started with this repository, you will need to have the following software installed on your machine:

- Java Development Kit (JDK) 8 or higher
- An Integrated Development Environment (IDE) like IntelliJ IDEA, Eclipse, or NetBeans

## Topics Covered

The repository is organized into the following core Java topics:

1. **Basic Syntax**
   - Variables and Data Types
   - Operators
   - Control Flow Statements (if, switch, loops)

2. **Object-Oriented Programming (OOP)**
   - Classes and Objects
   - Inheritance
   - Polymorphism
   - Encapsulation
   - Abstraction

3. **Data Structures**
   - Arrays
   - ArrayList
   - LinkedList
   - HashMap
   - HashSet

4. **Exception Handling**
   - Try-Catch-Finally
   - Throw and Throws
   - Custom Exceptions

5. **Streams and I/O**
   - File I/O
   - BufferedReader and BufferedWriter
   - InputStream and OutputStream

6. **Concurrency**
   - Threads and Runnable
   - Synchronization
   - Concurrency Utilities

7. **Collections Framework**
   - List, Set, and Map Interfaces
   - Iterator and ListIterator
   - Collections Utility Class

8. **Java 8 Features**
   - Lambda Expressions
   - Streams API
   - Optional Class

9. **Miscellaneous**
   - Generics
   - Annotations
   - Enumerations
   - Regular Expressions
